== Description ==

The VisaNet Direct Payment Gateway plugin integrates VisaNet Direct API with WooCommerce to handle direct payment transactions. It allows merchants to accept payments using VisaNet's payment processing solutions directly on their WooCommerce store.

= Features =
* Seamless VisaNet Direct payment integration with WooCommerce.
* Easy setup and configuration.
* Secure payment processing.
* Supports various currencies.

== Installation ==

1. Upload the `visanet-direct-payment` directory to the `/wp-content/plugins/` directory of your WordPress site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Settings > Payments.
4. Locate "VisaNet Direct Payment" and click on it to configure the settings.
5. Enter your VisaNet Merchant ID and API Key in the plugin settings.

== Configuration ==

1. **Merchant ID**: Enter your VisaNet Merchant ID.
2. **API Key**: Enter your VisaNet API Key.
3. **Title**: This is the title that users will see on the checkout page.
4. **Description**: This is a description that users will see on the checkout page.

== Usage ==

1. After configuration, the VisaNet Direct Payment option will be available on the WooCommerce checkout page.
2. Customers can select this payment option to complete their transactions.
3. The plugin will handle the payment processing and redirect customers to the VisaNet payment page.

== Frequently Asked Questions ==

= What is VisaNet Direct Payment? =
VisaNet Direct Payment is a service provided by VisaNet that allows merchants to process payments directly through their API. This plugin integrates that service with WooCommerce.

= Is this plugin compatible with all WooCommerce versions? =
This plugin is tested up to WooCommerce version 6.0. For compatibility with future versions, please refer to the plugin updates.
